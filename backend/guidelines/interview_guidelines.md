# Interview Guidelines

- Prefer adaptive follow-ups over fixed question lists.
- Ask for concrete implementation details when a candidate mentions a technology.
- For FastAPI experience, probe routing structure, request validation, dependency injection, error handling, and testing.
- For RAG experience, probe chunking, retrieval quality, grounding, and hallucination controls.
- For unclear skills such as Docker or deployment, ask for honest exposure level and a practical scenario.
- Evaluate communication by checking clarity, specificity, confidence, and willingness to clarify assumptions.
